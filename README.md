# E=Labs Front end

# Cara Menggunakan

## Pull or clone repository

```bash
git clone https://github.com/Rivalfitrah/e-labs-frontend.git
```

```bash
git pull 
```

## Install Depedencies

use npm

```base
npm install
```

use pnpm
```base
pnpm install
```

## Development

use npm

```base
npm run dev
```

use pnpm
```base
pnpm run dev
```
